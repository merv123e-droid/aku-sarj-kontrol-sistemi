# Otomatik Akü Şarj ve Koruma Sistemi

Bu projede XH-M603 Akü Şarj Kontrol Kartı kullanılarak otomatik akü şarj sistemi tasarlanmıştır.

## Kullanılan Donanımlar
- XH-M603 Akü Şarj Kontrol Kartı
- 12V Kurşun-Asit Akü
- 12V Adaptör
- Sigorta
- Bağlantı kabloları

## Sistem Özeti
Sistem akü voltajını sürekli izler. Akü dolduğunda şarjı otomatik keser. Voltaj düştüğünde tekrar şarj başlar.

## Bağlantı
12V Adaptör → XH-M603 → Akü

## Kullanım Alanları
- UPS sistemleri
- Kuluçka makineleri
- Güneş enerjisi sistemleri
- Karavan sistemleri
